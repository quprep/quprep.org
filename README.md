# quprep.org

Source for the [quprep.org](https://quprep.org) website.

Built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically build and deploy to GitHub Pages via GitHub Actions.

## Contributing

The actual library lives at [quprep/quprep](https://github.com/quprep/quprep). Do not open library-related issues here.
